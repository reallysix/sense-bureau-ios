# 传感器解释与产品边界

本文记录传感器页面的解释口径、算法决策和后续验收要求。面向用户的文案应保持简短，但不得超出这里定义的能力边界。

## 振动分析

### 可以做什么

- 比较桌面、家电或其他物体在不同状态下的相对振动强弱。
- 观察敲击、移动和周期性振动的短时趋势。
- 保存当前值、均方根和峰值，便于回看。

### 静止底噪处理

2026-07-22 真机反馈显示，手机静止时旧版历史曲线仍持续波动。旧算法把三轴加速度先转为幅度，再减去标量均值；方向性噪声无法正确抵消，图表自动缩放又放大了微小变化。

当前实现以校准样本建立三轴基线，用残差的第 95 百分位估算底噪门限，并把门限内的值归零。门限限制在 3–12 mg，图表量程下限为 10 mg。峰值取门限处理后的未平滑样本，以免过滤真实的短促震动。

真机验收：把手机稳定平放，等待约 1 秒完成基线采集；当前值和历史曲线应基本归零。轻敲桌面仍应立即出现峰值。若更换放置位置或姿态，应重新校准。

## 气压与相对高度

### 气压如何解释

- 1013.25 hPa 是标准海平面气压参考值，不是每个地点都必须达到的“人体正常值”。实际气压会随天气和海拔变化。
- App 的读数用于观察环境气压与变化趋势，不用于判断健康状况。
- 较快或较大的压力变化可能引起耳部或鼻窦不适，鼻塞时感受可能更明显；持续或明显不适应咨询医疗专业人员。

资料来源：

- [NOAA：Air Pressure](https://prod-01-alb-www-noaa.woc.noaa.gov/jetstream/atmosphere/air-pressure)
- [NOAA NDBC：Air Pressure](https://www.ndbc.noaa.gov/education/pressure.shtml)

### 相对高度如何解释

相对高度是根据气压变化估算的“相对于本次基准位置的垂直变化”。用户在地面点击“设为基准”后再上楼，可以估算上升高度；它不是海拔，也不能自动知道手机距脚下地面的高度。天气、空调、电梯增压、开关门和传感器漂移都会影响结果，因此它只适合趋势和近似比较。

## 磁场与水平仪

- 磁场探测用于比较手机周围磁场变化、寻找明显磁性干扰，不等同于专业金属探测器，也不能识别金属种类。
- 水平仪用于找平、比较坡度和归零后的相对角度；结果受保护壳、放置面和设备校准影响，不替代工程测量仪器。
