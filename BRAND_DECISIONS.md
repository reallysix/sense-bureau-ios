# 感知局品牌决策记录

状态：已确认  
确认日期：2026-07-19

## 最终结论

| 项目 | 结论 |
| --- | --- |
| 中文品牌名 | 感知局 |
| 英文品牌名 | Sense Bureau |
| GitHub 仓库 | `sense-bureau-ios` |
| 中文商店副标题建议 | 把 iPhone 变成随身实验室 |
| 英文商店副标题建议 | Pocket Sensor Lab |

## 决策依据

1. “感知”能够覆盖磁场、振动、水平/倾角、气压/高度等后续模块，不会把产品限制为金属探测器。
2. “局 / Bureau”具有轻量探索机构的品牌联想，比“工具箱”更年轻，又比纯卡通名称更适合默认科技皮肤。
3. 品牌名不承诺手机无法可靠实现的全金属识别能力；具体页面继续使用“磁场探测”等准确功能名。
4. 品牌与皮肤彼此独立。科技和卡通皮肤共享 Sense Bureau 品牌、测量事实与信息架构。
5. 初步公开检索没有发现明显的同名 App 或软件品牌，但这不是正式商标法律意见；提交 App Store 或进行商业推广前仍需完成目标市场商标与商店名称核验。

## 工程命名约定

- Xcode 工程、主 Target 与 Swift 模块：`SenseBureau`
- App 入口类型：`SenseBureauApp`
- Bundle ID：`com.sensebureau.ios`
- 测试 Target：`SenseBureauTests`、`SenseBureauUITests`
- 公开仓库：`sense-bureau-ios`
- 当前本地根目录 `power-app` 暂时保留，不作为品牌名称继续使用。

## 用户可见名称

- 简体中文系统下显示“感知局”。
- 英文及其他尚未本地化的系统下显示“Sense Bureau”。
- 功能页面仍使用准确的模块名称，例如“磁场探测 / Magnetic Field”。

## 公开源码边界

公开仓库包括应用源码、测试、工程配置、设计规范和第三方许可记录。以下内容不进入公开仓库：

- 带第三方品牌或作者标识的设计参考图片；
- 构建产物、测试结果包和截图证据；
- Xcode 用户状态、签名团队标识、证书和本地配置；
- 任何密钥、令牌或账号信息。

当前未选择开源许可证。公开仓库中的源码可公开阅读，但在明确许可证加入前不自动授予复制、修改或分发权利。
