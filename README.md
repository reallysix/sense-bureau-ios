# 感知局 · Sense Bureau

Sense Bureau 是一款把 iPhone 内置传感器变成随身实验工具的原生 iOS 应用。产品从磁场探测开始，计划逐步加入振动、水平/倾角与气压/高度等模块，并通过统一主题系统支持科技和卡通皮肤。

> 手机磁力计测量的是附近磁场变化，不能可靠识别所有金属，也不能替代专业安全或勘测设备。

## 当前进度

- 磁场三轴数据、综合场强、校准、滤波和峰值
- 首次测量引导、能力边界说明和校准准备清单
- 实时仪表、可调警报阈值与历史曲线
- 声音/触觉提醒开关，暂停、恢复和重新校准
- 磁场测量结果保存到 SwiftData
- 简体中文与英文即时切换及持久化
- Tech Signal 与 Playful Lab 两套可切换皮肤
- 实验室首页及实验室、磁场、设置三段导航
- 传感器页面生命周期管理与 SwiftData 本地记录模型
- 18 个单元测试与 7 个 UI 自动化测试

## 开发环境

- Xcode 26+
- iOS 18+
- Swift 6
- XcodeGen

生成工程并打开：

```bash
xcodegen generate
open SenseBureau.xcodeproj
```

模拟器可以验证界面与流程，但磁力计的真实行为必须在 iPhone 上验证。真机构建时请在 Xcode 中选择自己的开发团队。
